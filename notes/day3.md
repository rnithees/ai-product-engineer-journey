Day 3 Learning

- Pandas works with structured tabular data using DataFrames.
- A DataFrame organizes data into rows and named columns.
- Filtering selects rows directly using conditions.
- Feature creation builds new columns from existing columns.
- Sorting rearranges rows based on column values.
- describe() gives predefined statistics for numeric columns.
- apply() runs custom logic value by value.
- Pandas is useful for exploratory data analysis (EDA).
